┌──────────────────────────────────────────────────────────────────────────┐
│ PHÍM TẮT ĐIỀU KHIỂN & JOG                                                │
└──────────────────────────────────────────────────────────────────────────┘

  F1-F4        →  Chọn Port 1-4 (cả Jog và Teaching Mode)
  Q            →  Jog Plus (giữ để di chuyển +)
  A            →  Jog Minus (giữ để di chuyển -)
  X, Y, Z      →  Chọn trục X, Y, Z
  I, O, F      →  Chọn trục RI (Inner), RO (Outer), F (Focus)
  Space        →  Toggle Jog/Step Mode
  Ctrl+H       →  Hiển thị Help Form (hướng dẫn phím tắt)

┌──────────────────────────────────────────────────────────────────────────┐
│ QUY TRÌNH TEACHING NHANH                                                 │
└──────────────────────────────────────────────────────────────────────────┘

  1. Chọn Port (F1-F4)
  2. Bật Teaching Mode (radio button + password 1234)
  3. Jog đến vị trí (Q/A keys + chọn trục)
  4. Save điểm (Ctrl+số hoặc F5-F12)
     → Button chuyển màu xanh
  5. Test bằng Go (Ctrl+Shift+số hoặc Shift+F5-F12)

┌──────────────────────────────────────────────────────────────────────────┐
│ LƯU Ý QUAN TRỌNG                                                         │
└──────────────────────────────────────────────────────────────────────────┘

  ✓ Phím tắt CHỈ hoạt động trong Teaching Mode
  ✓ F1-F4 GIỮ NGUYÊN để chọn Port (không xung đột)
  ✓ Teaching points dùng Ctrl/Alt+số
  ✓ Mỗi Port có bộ teaching points riêng
  ✓ Button nhấp nháy VÀNG khi thực hiện
  ✓ Button Save chuyển XANH sau khi lưu thành công

┌──────────────────────────────────────────────────────────────────────────┐
│ PATTERN DỄ NHỚ                                                           │
└──────────────────────────────────────────────────────────────────────────┘

  SAVE   = Modifier + Số/F-key
  GO     = Modifier + Shift + Số/F-key

  1-4    = Tray operations (với Ctrl/Alt)
  F5-F12 = Socket & Camera (không cần số)
